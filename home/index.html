DataFan ──
打通从数据集成到数据应用的全流程!

DataFan 的初衷是为了打通企业从数据集成 -> 数据开发 -> 数据应用的全流程。将数据集成、数据开发（离线+实时）、数据可视化等平台开源出来，同开源社区一同治理和发展。
 GitHub Gitee

项目特性

质量可控

高质量的代码,后端代码通过严格的checkstyle进行代码规范检查,前端代码全部经过eslint语法的严格检验,保证可读性和健壮性

简单易用

DataFan 的初衷是为了打通企业从数据集成->数据开发->数据应用的全流程,所有项目的设计目标就是为了能开箱即用,简化开发和使用

文档详尽

所有项目都会有对应的帮助文档,每个使用细节和注意事项都详细罗列,并辅助大量代码示例,让开发者一目了然

健壮稳定

数据行业专业团队打造,每个API,每处设计都经过精心打磨,测试用例齐全,经历线上项目验证,保证稳定可靠

多引擎

数据开发支持多引擎自由选择，Local、Spark、Flink。

一站式解决方案

通过数据集成 -> 数据开发 -> 数据可视化等项目的组合，整体功能涵盖了从数据同步到数据加工到数据使用的全流程,是一个一站式综合数据平台
